# DevOps Application Deployment (Guvi Project 3)
DevOps project:
Dockerize React app from GitHub repo,
push to Docker Hub (dev/public,
prod/private repos),
setup Jenkins CI/CD (dev branch -> dev repo,
master merge -> prod repo), 
deploy to AWS EC2 t2.micro on port 80, 
configure security group (HTTP world access, 
SSH your IP only), 
add open-source monitoring with alerts, 
submit GitHub URL + live site URL + Docker images + screenshots of Jenkins/AWS/Docker Hub/site/monitoring.

5 steps:
Docker build + Docker Hub push
AWS EC2 launch + deploy
Jenkins pipeline + GitHub webhook
Uptime monitoring
Screenshots + submit
